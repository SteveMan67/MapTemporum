# MapTemporum - Historical Map
(name to be replaced once I think of a better one)

Serves as an interface between all the good data on openhistoricalmap.org and teachers/students. Currently includes features:  
 - An interactive map of history
 - User custom markers
 - Wikipedia article link on right click
 - Filtering the map (partial support)
 - Saving current map as a jpg or png (with transparency)
